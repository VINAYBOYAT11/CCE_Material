## 🚀 How to Use

### Compile & Run C Programs

```bash
# Clone the repository
git clone https://github.com/VINAYBOYAT11/CCE_Material.git
cd CCE_Material

# Compile a C file
gcc filename.c -o output

# Run the output
./output      # Linux/Mac
output.exe    # Windows
```

### View PDF Materials
Simply navigate to the respective folder and open the PDF file with any PDF viewer.
